# TodoList 
Affichez la mise en page optimale de l'application en fonction de la taille de l'écran de leur appareil
- Voir les états de survol pour tous les éléments interactifs de la page
- Ajouter de nouvelles tâches à la liste
- Marquer todos comme terminé
- Supprimer les todos de la liste
- Filtrer par tous / tâches actives / complètes
- Effacer toutes les tâches terminées
- Basculer entre les modes clair et sombre
- ** Bonus **: Glissez-déposez pour réorganiser les éléments de la liste
